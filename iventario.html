Sistema de Gestión de Inventario

Proyecto desarrollado en C# utilizando aplicaciones de consola.

>> Descripción del Proyecto

Este proyecto consiste en un Sistema de Gestión de Inventario desarrollado en C#. Su propósito es permitir la administración de productos mediante operaciones de registro, consulta, modificación, venta y eliminación de datos almacenados en archivos de texto.

Demostración del Proyecto

>> Objetivo

Aplicar los conocimientos adquiridos durante el curso de programación mediante el desarrollo de una aplicación capaz de gestionar productos, controlar existencias y almacenar información de manera persistente utilizando archivos.

>> Funcionalidades Principales

  • Agregar productos al inventario.
  • Visualizar productos registrados.
  • Modificar existencias.
  • Registrar ventas.
  • Eliminar productos específicos.
  • Borrar completamente el inventario.
  • Guardar información en archivos de texto.
  • Recuperar datos almacenados al iniciar el programa.

Herramientas usadas

Tecnología Uso
C# Lógica principal del sistema
Visual Studio Entorno de desarrollo
Aplicación de Consola Interfaz de usuario
Archivos TXT Almacenamiento de datos

  • Estructuras (Struct).
  • Arreglos.

>> Lo que aprendi al desarrollar el proyecto

Aprendí a trabajar con estructuras de datos, manipular archivos para almacenar información de forma permanente y desarrollar aplicaciones capaces de gestionar procesos reales mediante menús interactivos.
